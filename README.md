# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, NLP, and PDF processing techniques. This project helps students, freshers, and job seekers evaluate their resumes by extracting skills, calculating an ATS score, recommending job roles, and providing improvement suggestions.

---

## Features

✅ Upload Resume PDF

✅ Extract Resume Content

✅ Detect Technical Skills

✅ Extract Email & Phone Number

✅ Identify Education Details

✅ Estimate Experience Level

✅ Calculate ATS Score

✅ Recommend Suitable Job Roles

✅ Generate Resume Improvement Suggestions

---

## Technologies Used

* Python
* SpaCy
* PyPDF
* Regular Expressions (Regex)
* Google Colab

---

## Project Workflow

1. Upload Resume PDF
2. Extract Text from PDF
3. Clean and Process Text
4. Detect Skills and Keywords
5. Extract Contact Information
6. Analyze Education & Experience
7. Calculate ATS Score
8. Recommend Job Roles
9. Generate Suggestions

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/AI-Resume-Analyzer.git
```

Install dependencies:

```bash
pip install pypdf spacy
python -m spacy download en_core_web_sm
```

Run the notebook or Python script.
